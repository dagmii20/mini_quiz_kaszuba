# Mini Quiz

## Opis

Prosta aplikacja quizowa na Androida.  
Użytkownik odpowiada na pytania i na końcu wyświetlany jest wynik.

## Funkcje

- 5 pytań w quizie  
- pytania losowane z puli 10 pytań  
- 3 odpowiedzi do wyboru  
- zliczanie punktów  
- przycisk start i reset  

## Jak uruchomić

Sklonuj repozytorium, wpisując w terminalu:  
```bash
git clone [https://github.com/ICHACZ3000/mini-quiz-Brzeski.git](https://github.com/ICHACZ3000/mini-quiz-Brzeski.git)
```

Otwórz projekt w Android Studio i uruchom aplikację.

## Pliki

- MainActivity.java  
- Question.java  
- activity_main.xml
